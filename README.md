# Spotify Clone Music Player

A Spotify-inspired Music Player built using HTML, CSS, and Vanilla JavaScript. The project provides a responsive music streaming interface with dynamic playlist loading, playback controls, keyboard shortcuts, and an interactive user experience.

## Features

- Dynamic playlist loading from folders
- Play and Pause functionality
- Next and Previous song controls
- Music progress bar with seek functionality
- Volume control slider
- Responsive design for desktop and mobile devices
- Dynamic album and song rendering
- Real-time song information display
- Keyboard shortcuts support

### Keyboard Shortcuts

| Key | Action |
|------|----------|
| Spacebar | Play / Pause |
| N | Next Song |
| P | Previous Song |

Spacebar - Play and Pause
N - Next song
P - Previous song
